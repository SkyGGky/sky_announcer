La instalacion es sencilla, crea una carpeta llamada [scripts] y pon en el server.cfg el comando:

ensure [scripts]

Esto hara que el servidor inicie tal carpeta, y hay dentro pondras el script en bruto.

Para modificarlo es sencillo, vete a la carpeta Client y hay hay un archivo .lua

Modificalo a tu gusto y ya tienes tu script de anuncios!

--------------

The installation is simple, create a folder called [scripts] and put the command in the server.cfg:

ensure [scripts]

This will make the server start such a folder, and inside you will put the raw script.

To modify it is simple, go to the Client folder and there is a .lua file

Modify it to your liking and you already have your ad script!
